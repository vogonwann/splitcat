# Splitcat

Simple linux tool to split and merge files.

## Getting Started

Head over to [releases](https://github.com/vogonwann/splitcat/releases) and grab the version according to your distro.

Works on:
- Desktops ️ 🖥️
- Laptops 💻️
- Linux Phones 📱

Android 🤖 and Windows 🪟 versions are coming soon.
